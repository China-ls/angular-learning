### 模块机制

- 把各功能模块的具体实现代码独立出来
- module机制作为一个壳子，对功能模块进行包装
- 每个功能分组，使用一个总的壳子来包装，减少上级模块的引用成本
- 把每个壳子文件把module的name属性export出去
